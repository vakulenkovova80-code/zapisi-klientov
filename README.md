# 💗 Записи клиентов — макияж и причёски

Приложение для телефона, в котором мастер ведёт записи клиентов: имя, телефон,
дата и время, услуга и цена, заметки и фото. Работает как обычное приложение на
iPhone (иконка на экране «Домой»), **офлайн**, а все данные хранятся прямо в
телефоне. Сервер не нужен.

## Что умеет
- 📋 Список ближайших записей по дням
- 🗓️ Календарь с отметками дней + **свободное время дня** (🟢 «Свободно: 8:00–13:30…») в рамках рабочих часов
- ➕ Новая запись (имя, телефон, дата, услуга, цена, **длительность**, заметка, фото) + кнопка «В Календарь iPhone» (надёжное напоминание)
- 👥 **База клиентов**: добавление напрямую, телефон, «откуда пришёл» (источник), заметка, поиск, редактирование, история визитов
- ✉️ **Рассылка**: по всем или по источнику — WhatsApp по каждому клиенту + копирование номеров/текста
- ⚙️ Свои услуги и цены, **рабочие часы**
- 💾 Резервная копия (сохранить/восстановить файлом) — **делайте раз в неделю!**

## Как опубликовать (делается один раз)

Нужен компьютер только для этого первого шага. После публикации приложение живёт
в интернете само и от компьютера не зависит.

### 1. Залить код на GitHub
```bash
# из папки проекта, один раз:
gh repo create zapisi-klientov --private --source=. --push
```
(или создайте репозиторий на github.com вручную и запушьте ветку `main`).

### 2. Подключить Netlify (бесплатно)
1. Зайдите на [netlify.com](https://netlify.com), войдите через GitHub.
2. **Add new site → Import from Git →** выберите репозиторий `zapisi-klientov`.
3. Настройки сборки подхватятся автоматически из `netlify.toml`
   (`npm run build`, папка `dist`). Нажмите **Deploy**.
4. Netlify выдаст постоянный адрес, например `https://zapisi-klientov.netlify.app`.
   Это и есть приложение.

### 3. Установить на iPhone (делает мастер)
1. Откройте адрес в **Safari**.
2. Нажмите кнопку «Поделиться» → **«На экран Домой»**.
3. Появится иконка 💗 «Записи». Открывайте её как обычное приложение.

## Локальная разработка
```bash
npm install      # установить зависимости
npm run dev      # запустить локально
npm test         # прогнать тесты
npm run build    # собрать продакшн-версию в dist/
```

## Важно про данные
Данные хранятся только в этом телефоне. Если телефон потеряется или его сбросят —
записи пропадут. Поэтому в разделе **Настройки → Резервная копия** регулярно
(раз в неделю) нажимайте «Сохранить копию» и храните файл в «Файлы»/iCloud.
